# STATS-507-Final-Project-Repository

This repository contains all materials for Yanran Chen’s Final Project in **STATS 507** at the University of Michigan.

## Repository Contents

### `STATS_507_Final_Project_Report_IEEE.pdf`
Final project report written in IEEE conference format.

### `final_proj.ipynb`
Jupyter Notebook containing the full codebase:
- Data loading and cleaning  
- Exploratory data analysis (EDA)  
- Feature engineering  
- Model training (Linear Regression, Random Forest, XGBoost)  
- Model evaluation (MAE, RMSE, R²)  
- Feature importance visualizations  

### `used_cars.csv`
Original dataset of used cars (downloaded from Kaggle). Includes vehicle attributes and sale price information.

### `README.md`
Overview of the repository and file structure.

---

## Notes
All results, figures, and modeling steps shown in the final report can be reproduced using `final_proj.ipynb`.
